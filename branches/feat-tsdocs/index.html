    @nillion/secretvaults

    A TypeScript client library for building with Nillion Private Storage, a decentralized storage system keeps sensitive data secret by storing encrypted shares across a cluster of nilDB nodes. Each nilDB node stores a separate share of the encrypted data, ensuring no single node can reveal the original value.

    Installation

    Install the package via npm:

    npm install @nillion/secretvaults

    Or using yarn:

    yarn add @nillion/secretvaults

    Or using pnpm:

    pnpm add @nillion/secretvaults
